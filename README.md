Headline Hub is a Python desktop application that fetches and displays the latest news headlines using a REST API. The app provides an easy-to-use interface for browsing news articles, viewing images and summaries, and opening full articles in a web browser.

FEATURES
Displays live top headlines using NewsAPI
Shows news title, description, and image
Navigate through articles using Previous and Next buttons
Opens full news articles in the default web browser
Handles missing images and API errors gracefully
Simple and clean Tkinter-based GUI

TECHNOLOGIES USED
Python
Tkinter
NewsAPI
Requests
Pillow


LEARNING OUTCOMES
Working with REST APIs in Python
Parsing JSON data
Building desktop applications using Tkinter
Loading and displaying images from URLs
Handling runtime and network errors
